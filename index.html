VeeCode Claude Gateway

Corporate gateway for Claude Code: Google identity in, attributable spend + audit logs out.

Install (macOS / Linux)

curl -fsSL https://veecode-claude-gateway.github.io/install.sh | bash

Install (Windows PowerShell 7+)

iwr https://veecode-claude-gateway.github.io/install.ps1 | iex

What this does

Installs a thin claude wrapper that turns your active gcloud identity into a short-lived gateway virtual key, points Claude Code at the gateway, and execs the real binary. The full developer onboarding doc lives in the meta-repo.

Source: veecode-claude-gateway.github.io. Architecture: claude-gateway-parent.
